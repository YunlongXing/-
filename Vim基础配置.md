# Vim基础配置

```$ vim ~/.vimrc```

```vim
set number
set tabstop=4
set shiftwidth=4
syntax on
set showmode
set showcmd
set mouse=a
set encoding=utf-8
set t_Co=256
filetype indent on
set cursorline
set wrap
set ruler
set showmatch
set hlsearch
set incsearch
set spell spelllang=en_us
set wildmenu
set wildmode=longest:list,full
```
